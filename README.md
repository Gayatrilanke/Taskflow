# 🚀 TaskFlow - MERN Stack Task Management Web Application

## 📌 Project Description
TaskFlow is a full-stack task management web application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). It allows users to efficiently manage daily tasks, track progress, and improve productivity through a user-friendly dashboard.

---

## 🛠️ Tech Stack Used

### 🔹 Frontend
- React.js
- HTML
- CSS
- JavaScript

### 🔹 Backend
- Node.js
- Express.js

### 🔹 Database
- MongoDB

---

## ✨ Features

- User Registration & Login Authentication
- Create, Update & Delete Tasks
- Project Management
- Dashboard for Task Monitoring
- Secure API Integration
- Responsive UI Design
